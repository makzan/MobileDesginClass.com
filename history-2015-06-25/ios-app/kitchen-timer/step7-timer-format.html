Step 7—Timer Format

  1. Create a new file named NSNumber+MZTimerFormat.swift.

    import UIKit

extension NSNumber {
  func stringOfMinutesSeconds () -> NSString{
    if (integerValue < 0) {
      return "00:00"
    }
    let m = integerValue / 60
    let s = integerValue % 60
    return NSString(format: "%02d:%02d", m, s)
  }
}

  2. In the tick method, we change to the following code:

    func tick() {
  let remainSeconds:NSNumber = MZTimer.sharedTimer.remainSeconds()

  countDownLabel.text = remainSeconds.stringOfMinutesSeconds() as String
}
  3. Try to run the app. Add 30 seconds and press start. We can now see the timer label is counting down.

  4. We can improve the tick method. Replace the tick method with the following code:

    func tick() {
  // only tick the UI timer when the MZTimer is running for any local notification.
  if (!MZTimer.sharedTimer.isRunning()) {
    countDownLabel.text = (seconds as NSNumber).stringOfMinutesSeconds() as String
    showPrepareUI()
    return
  }

  showRunningUI()

  let remainSeconds:NSNumber = MZTimer.sharedTimer.remainSeconds()

  countDownLabel.text = remainSeconds.stringOfMinutesSeconds() as String
}

By the way

  • Swift automatically bridges Double, Integer, Bool into NSNumber. But it won’t automatically cast it.

  • We have cast the NSString into String for UILabel text.

  • Extensions is similar to Category in Objective-C.
