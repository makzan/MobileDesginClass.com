Step 8—Admin

The admin is a tab bar controller that contains 2 table view controllers: NewSecretsViewController and ApprovedSecretsViewController.

  1. Create a new target and initialize the Parse library in the AppDeletage class.
  2. Make the application a tab bar controller. It contains 2 table view controllers, which are NewSecretsViewController and ApprovedSecretsViewController.
  3. Enable the Refresh Control in both table view controller.

  4. In the viewDidLoad function, we register the refresh control’s event.

    self.refreshControl?.addTarget(self, action: "refresh", forControlEvents: UIControlEvents.ValueChanged)

  5. Then we implement the refresh function in the NewSecretsViewController class:

    func refresh() {
  Secrets.sharedSecrets.allNonApprovedSecrets { (entries) -> Void in
    self.entries = entries
    self.tableView.reloadData()
  }
}
  6. In the ApprovedSecretsViewController class, we implement the same refresh function. But we call the allApprovedSecrets function instead.

  7. For both view controllers, we implement the following table view data source functions:

    override func tableView(tableView: UITableView, numberOfRowsInSection section: Int) -> Int {
  self.entries.count
}

override func tableView(tableView: UITableView, cellForRowAtIndexPath indexPath: NSIndexPath) -> UITableViewCell {
  var cell = tableView.dequeueReusableCellWithIdentifier("cell")
  if cell==nil {
    cell = UITableViewCell(style: .Default, reuseIdentifier: "cell")
  }

  cell?.accessoryType = .None
  if entries[indexPath.row]["is_public"] as! Bool == true {
    cell?.accessoryType = .Checkmark
  }

  cell?.textLabel?.text = entries[indexPath.row]["content"]

  return cell!
}

  8. When any row is selected, we toggle the approval state. The NewSecretsViewController class calls the approveSecret function and the ApprovedSecretsViewController class calls the unapproveSecret function. The other part is the same.

    override func tableView(tableView: UITableView, didSelectRowAtIndexPath indexPath: NSIndexPath) {
  tableView.deselectRowAtIndexPath(indexPath, animated: true)

  Secrets.sharedSecrets.approveSecret(entries[indexPath.row])

  tableView.reloadData()
}

  9. In the NewSecretsViewController class, we allow deleting the table view cell by the following delegate function:

    override func tableView(tableView: UITableView, commitEditingStyle editingStyle: UITableViewCellEditingStyle, forRowAtIndexPath indexPath: NSIndexPath) {
  if editingStyle == .Delete {
    Secrets.sharedSecrets.deleteSecret(entries[indexPath])

    let newEntries = NSMutableArray(array: entries)
    newEntries.removeObjectAtIndex(indexPath.row)
    entries = newEntries

    tableView.deleteRowsAtIndexPaths(indexPath, withRowAnimation: .Fade)
  }
}
