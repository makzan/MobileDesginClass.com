Pragma mark in swift

The Pragma Mark in Objective-C:

#pragma mark

The Pragma Mark in Swift:

// MARK:

For example, the horizontal line can be presented by:

// MARK: -

Title following an horizontal line would be:

// MARK: - Text Goes Here
