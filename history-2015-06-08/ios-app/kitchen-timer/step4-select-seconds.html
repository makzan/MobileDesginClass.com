Step 4—Select seconds to schedule

  1. Add 2 IBActions. Link the touchUpInside event of buttons of +30s and -30s to add30s and minus30s method.

  2. Let’s move to ViewController.swift file. Add the seconds instanc variable to the class.

    var seconds:NSTimeInterval = 0
  3. Implement the IBActions that manipulate the seconds value.
    // MARK: - Select seconds
@IBAction func add30s(sender: AnyObject) {
  seconds += 30
}

@IBAction func minus30s(sender: AnyObject) {
  seconds = max(seconds-30, 0) // minus 30, minimun 0.
}

  4. We originally hard-coded the notification time interval to 30 seconds. Now we change the hard-coded value by using the instant variable seconds.
    The old code:

    @IBAction func startTimer(sender: AnyObject) {
  showRunningUI()
  LocalNotificationManager.sharedManager.scheduleNotificationWithInterval(30.0)
}

    Change to the following new code:

    @IBAction func startTimer(sender: AnyObject) {
  showRunningUI()
  LocalNotificationManager.sharedManager.scheduleNotificationWithInterval(seconds)
}
