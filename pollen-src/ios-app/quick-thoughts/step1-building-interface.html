Step 1—Building Interface

It’s a simple interface that we have an UITextView and UIButton.

For the navigation bar, we use the UINavigationController and set this view controller as its rootViewController

After this step, you should have the following IBOutlet created.

@IBOutlet weak var mainTextView: UITextView!
@IBOutlet weak var bottomMargin: NSLayoutConstraint!
@IBOutlet weak var saveButton: UIButton!

bottomMargin is the distance constraint of save button’s buttom to view controller’s bottom edge.
