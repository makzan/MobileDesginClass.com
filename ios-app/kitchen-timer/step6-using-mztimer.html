Step 6—Using MZTimer

We implemented the MZTimer and MZMutableTimer, now we use it in our application.

  1. We connect the label into code as countDownLabel IBOutlet.

    @IBOutlet weak var countDownLabel: UILabel!

  2. Inside viewDidLoad.

    if (MZTimer.sharedTimer.isRunning()) {
  showRunningUI()
} else {
  showPrepareUI()
}

  3. For the startTimer and cancelTimer, we prepare the user interface and setup the timer via the MZMutableTimer class.

    @IBAction func startTimer(sender: AnyObject) {
  showRunningUI()
  MZMutableTimer.sharedMutableTimer.scheduleAlertForSeconds(seconds)
}

@IBAction func cancelTimer(sender: AnyObject) {
  showPrepareUI()
  MZMutableTimer.sharedMutableTimer.cancelAlerts()
}

  4. Then we define a NSTimer that change the interface constantly. Note that this logic only affects the interface. The actually counting logic is done via the MZTimer and local notification.

    // MARK: - Timer interface
func startUITimer() {
  let timer = NSTimer.scheduledTimerWithTimeInterval(0.2, target: self, selector: "tick", userInfo: nil, repeats: true)
}

func tick() {
  let elapsedSeconds = MZTimer.sharedTimer.elapsedSeconds()

  NSLog("%02.0f", elapsedSeconds)

  countDownLabel.text = NSString(format: "%02.0f", elapsedSeconds) as String

  // TODO: make the counting format nicer.
}
