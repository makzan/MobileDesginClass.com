Step 1

In this step, we create the basic user interface.

  1. Download the images.

  2. Crate new XCode project, with swift and universal option

  3. Setup the interface like the following.

  4. Create outlet.

    @IBOutlet weak var startButton: UIButton!
    @IBOutlet weak var stopButton: UIButton!

  5. Define 2 functions that controls the interface visibility.

    // MARK - UI Visibility
    func showRunningUI() {
        stopButton.hidden = false
        startButton.hidden = true
    }
    func showPrepareUI() {
        stopButton.hidden = true
        startButton.hidden = false
    }

  6. Then we define the following button handler.

    @IBAction func startTimer(sender: AnyObject) {
        showRunningUI()
    }
    @IBAction func cancelTimer(sender: AnyObject) {
        showPrepareUI()
    }

  7. At last, we setup the interface at the viewDidLoad
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view, typically from a nib.
        showPrepareUI()
    }
